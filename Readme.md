Ai dot
==============
Simple AI framework for Go.

Requirements
-----------------
    go 1.19+
    
    option:
    Graphviz(dot):draw model graph and loss graph

Installation
-----------------
    
    go get github.com/easedot/aidot
    
    option:
    brew install graphviz

Examples
-----------------

    go build examples/example_42.go

    go build examples/example_...go
